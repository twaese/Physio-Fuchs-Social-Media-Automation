# Einwilligungserklärungen – Vorlagen

Die folgenden Texte sind **Vorlagen**. Vor dem produktiven Einsatz
**zwingend** durch die zuständige Stelle (Datenschutzbeauftragte:r,
ggf. Anwält:in) prüfen lassen und auf den konkreten Praxis-Kontext
anpassen.

Unterschriebene Originale **nicht** in diesem Repo speichern, sondern
in einem geschützten Praxis-Ordner. Hier nur Versionierung der Vorlagen.

---

## 1. Mitarbeiter:innen – Bild- und Tonaufnahmen für Social Media

```
Einwilligung in die Veröffentlichung von Bild- und Tonaufnahmen
__________________________________________________________________

Name:                  ____________________________________
Funktion in der Praxis:____________________________________
Datum der Einwilligung:____________________________________

Ich, die oben genannte Person, willige hiermit ausdrücklich ein,
dass die Physiotherapiepraxis Physio Fuchs (im Folgenden "Praxis")

[ ] Fotos
[ ] Videos
[ ] Tonaufnahmen
[ ] aus Trainings-/Behind-the-Scenes-Material

von mir anfertigt und für folgende Zwecke verwendet:

[ ] Veröffentlichung auf Instagram
[ ] Veröffentlichung auf Facebook
[ ] Veröffentlichung auf TikTok / YouTube Shorts (nur falls aktiv)
[ ] Verwendung in der praxiseigenen Website
[ ] Verwendung in praxiseigener Print-Werbung
[ ] Verwendung in internen Schulungsunterlagen

Die Aufnahmen können bearbeitet werden (Schnitt, Farbkorrektur,
Untertitel, Hinzufügen von Praxis-Logo o. ä.).

Mir ist bekannt, dass

- die Einwilligung freiwillig ist und ich sie jederzeit ohne Angabe von
  Gründen mit Wirkung für die Zukunft widerrufen kann
  (per E-Mail an: ____________________________).
- bereits veröffentlichte Inhalte nach Widerruf von praxiseigenen
  Kanälen entfernt werden, eine vollständige Entfernung aus dem
  Internet (Reposts, Cache, Drittseiten) jedoch nicht garantiert
  werden kann.
- die Inhalte ggf. auf Servern außerhalb der EU verarbeitet werden
  (insbesondere durch Meta Platforms).

Ort, Datum: ___________________________________________________
Unterschrift:__________________________________________________
```

## 2. Avatar-/KI-Sprecher:in (Judith Fuchs)

```
Einwilligung zur Erstellung und Nutzung eines KI-basierten Avatars
__________________________________________________________________

Name: Judith Fuchs
Funktion: Inhaberin Physio Fuchs
Datum: ____________________________________

Ich, Judith Fuchs, willige hiermit ausdrücklich ein, dass

[ ] Bild- und Videoaufnahmen meiner Person
[ ] Tonaufnahmen meiner Stimme

zum Zweck der Erstellung eines KI-basierten Avatars und/oder eines
Voice-Clones verwendet werden. Anbieter / verwendete Tools:

   ____________________________________________________________
   (z. B. HeyGen, Synthesia, ElevenLabs)

Der Avatar wird ausschließlich für folgende Zwecke eingesetzt:

[ ] Social-Media-Inhalte der Praxis Physio Fuchs
[ ] Erklär-/Aufklärungsvideos auf praxiseigenen Kanälen
[ ] interne Schulungsunterlagen

Der Avatar wird in den veröffentlichten Inhalten als KI-generiert
gekennzeichnet (z. B. mit Hinweis "🤖 Digitaler Avatar" o. ä.).

Mir ist bekannt:

- die Einwilligung ist freiwillig und jederzeit widerrufbar.
- nach Widerruf werden Avatar und Voice-Clone beim Anbieter gelöscht,
  bereits veröffentlichte Inhalte werden zeitnah von praxiseigenen
  Kanälen entfernt.
- Trainings- und Voice-Clone-Daten werden in einem geschützten Bereich
  gespeichert und nicht für Trainings Dritter freigegeben.
- für den Anbieter besteht ein Auftragsverarbeitungsvertrag (AVV).

Ort, Datum: ___________________________________________________
Unterschrift:__________________________________________________
```
