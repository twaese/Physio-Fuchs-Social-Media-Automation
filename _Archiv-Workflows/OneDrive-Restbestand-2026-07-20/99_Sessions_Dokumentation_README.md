# Session-Archiv

Hier liegen die Zusammenfassungen und To-Do-Listen vergangener
Arbeits-Sessions. Sie dienen als Kontext-Anker für künftige Sessions
und als historische Dokumentation der Architektur-Entscheidungen.

**Aktuellste Session-Notiz** liegt immer im Repo-Root unter
`SESSION-Naechste-Schritte_<Datum>.md` und wird beim Start der
nächsten Session zuerst gelesen.

---

## Konvention

| Dateityp                                 | Zweck                                           |
| ---------------------------------------- | ----------------------------------------------- |
| `SESSION-Zusammenfassung_<Datum>.md`     | Was wurde in dieser Session erreicht?           |
| `SESSION-Naechste-Schritte_<Datum>.md`   | Was steht für die nächste Session an?           |

Ältere Notizen wandern hierher, sobald eine neue Session-Notiz im Root
liegt.
